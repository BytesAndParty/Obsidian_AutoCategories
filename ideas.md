# Feature Ideas

## Category Tree View in der Sidebar

Eigene Sidebar-Ansicht ähnlich wie Tag Wrangler:

- Neue View mit `registerView()` registrieren
- Hierarchische Tree-Darstellung für verschachtelte Kategorien
- Rechtsklick-Kontextmenü mit Aktionen:
    - Umbenennen
    - Löschen
    - Zusammenführen (Merge)
    - In Hierarchie verschieben
- Icon in der linken Sidebar
- State-Management für expandierte/kollabierte Nodes
- Drag & Drop für Neuordnung

**Voraussetzungen:**

- Verständnis der Obsidian ItemView API
- Custom CSS für Tree-Darstellung
- Event-Handling für Kontextmenü

**Referenz:** Tag Wrangler Plugin als Inspiration
